Detailed Description

输入两个整数A B ,range in [-100 ,- 100]，若A B都小于 0，则输出 All less than 0 ；否则就判断A B中的最大值y = max(A,B), y是否处于[25 ,65] 之间，若处于，则输出 25 <= max(A,B) <= 65，否则输出 Illegal !
